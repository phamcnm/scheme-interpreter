# scheme-interpreter
Scheme interpreter in C for CS251. 

## How to run
To run the interpreter on the tests in the test_file_folders, type in command-line:
without valgrind:
  python3 test_m.py 1
with valgrind
  python3 test_m.py
To run with a specific test

valgrind ./interpreter < myprog.scm
